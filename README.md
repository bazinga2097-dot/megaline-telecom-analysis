# Video Game Sales Analysis

Exploratory Data Analysis project developed in Python using Pandas and NumPy to analyze global video game sales trends.

## Objectives

* Clean and transform sales data
* Analyze trends by genre and region
* Visualize sales behavior
* Identify market tendencies using statistical analysis

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook

## Main Insights

* Action and Sports genres showed the highest global sales.
* Regional preferences differed significantly between North America, Europe and Japan.
* Historical trends suggested increasing dominance of multiplayer and online-focused genres.
